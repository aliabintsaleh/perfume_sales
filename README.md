# 🧪 Perfume Sales Analysis – Power BI Dashboard

## 📊 Overview

An interactive **Power BI dashboard** analyzing perfume sales, inventory performance, and operational KPIs for an e-commerce perfume retailer.

**Objective:** Identify sales drivers, optimize inventory levels, detect stale listings, and provide actionable business recommendations.

**Time Period:** ~1 year of sales data (2024)
**Total Listings Analyzed:** 845 products  
**Total Units Sold:** 489,364  
**Current Inventory:** 17,647 units  

---

## 🎯 Key Performance Indicators (KPIs)

| KPI | Value | Status |
| :--- | :--- | :--- |
| **Total Units Sold (Annual)** | 489,364 | ✅ Excellent |
| **Total Available Stock** | 17,647 | 🔴 Very Low |
| **Stock to Sales Ratio** | 4% | 🔴 Unhealthy |
| **Inventory Turnover Rate** | 27.7x / year | 🟡 Very Fast |
| **Total Inventory Value** | ~SAR 2.65M | 💰 Capital tied up |
| **Conversion Rate (Overall)** | 96.5% | 🔴 Critical (Stockout risk) |
| **Top 5 Brands Share** | 91% of total sales | 🟠 High concentration |

---

## 🔍 Key Insights

### 1. Inventory Does NOT Keep Up with Demand 🔴
- Current stock covers only **13 days** of sales
- 96.5% conversion rate means almost everything that enters inventory gets sold
- **Critical risk:** Stockout within two weeks

### 2. High Brand Concentration 🎯
- **Top 5 brands** account for 91% of all sales:
  1. calvin klein – 47K units (34%)
  2. vercace – 32K units (22%)
  3. elizabeth tayior – 21k units (15%)
  4. vera wang – 20k units (14%)
  5. burberry – 19k units (13%)
- This creates **supplier dependency risk**

### 3. EDP Outperforms EDT in Every Metric 💎
| Metric | EDP | EDT | Difference |
| :--- | :--- | :--- | :--- |
| Avg Sales Volume | 450 | 1003 | 
| Avg Price | $43 | $33 ||
| Revenue Share | 69% | 31% 
- **Recommendation:** Focus on EDP for higher margin

## ⚠️ Risk Assessment

| Risk | Level | Impact |
| :--- | :--- | :--- |
| 91% reliance on 5 brands | 🟠 Medium | Supply chain vulnerability |
| Stale listings with active stock | 🟡 Low | Wasted capital |
| Products with <10% conversion | 🟡 Low | Frozen inventory |

## 🚀 How to Run This Project*

1. **Download** the `perfume_dashboard.pbix` file
2. **Open** with Power BI Desktop (free download from Microsoft)
3. **Update data source** if your CSV file location is different
   - Go to `Home` → `Transform Data` → `Data source settings`
4. **Refresh** the data (`Home` → `Refresh`)
